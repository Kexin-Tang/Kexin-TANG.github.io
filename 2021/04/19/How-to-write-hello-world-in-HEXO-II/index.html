Post your file

Now, you can begin to write your markdown or ejs files. When you want to post a file, you can just run codes in orders.

  1. hexo new post <fileName>
  2. write something
  3. hexo clean
  4. hexo g
  5. hexo d

Front-matter

When you new a file, HEXO will automatically generate a file in certain format, it depends on your METHOD (like post, draft and page) in scaffolds.

categories & tags

Except title and date, there are many other key words:

  • categories – indicates the categories it belongs to
  • tags – indicates the tags it has

Notice: categories is different with tags. tags is unordered, while categories is ordered.

This file will be attached tags with “web” and “hexo”, these tags are parallel. However, it also will be archived to Web and HEXO, and Web is father category, HEXO is child category.
