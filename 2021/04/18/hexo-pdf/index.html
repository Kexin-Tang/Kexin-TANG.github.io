If you want to insert .pdf files in your HEXO pages, there are two methods.

  • METHOD 0

You can install HEXO package hexo-pdf.

Then, you can use .ejs syntax {% pdf path %} to achieve function.

  • METHOD 1

In your index.md, you can use html syntax to achieve this function.

You can adjust the width and height even other properties.
