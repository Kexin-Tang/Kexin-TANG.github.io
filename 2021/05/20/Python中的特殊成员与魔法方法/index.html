Python中的特殊成员与魔法方法

特殊成员和魔法方法是一系列特定结构的成员/方法，其格式为__XXX__，使用这些成员和方法通常会有比较奇特的效果，比如说 使用某个函数时就会自动调用某个方法，想要查看隐藏的信息就要访问特殊的成员 等。

特殊成员

__class__

用于查看类名

__module__

用于查看类的来源

__dict__

用于查看属性

__str__

定义打印输出时的显示内容

魔法方法

__new__(cls, ...) 和 __init__(self, ...)

注意，__new__(cls, ...)为构造函数，用于创建类对象，而__init__(self, ...)仅仅是初始化函数，用于将预设的内容传递到新建的类内。这两个函数在使用诸如myClass = MyClass()时自动调用。

__call__(self, *args, **kwargs)

使得类和函数一样，可以直接加上()并传入参数调用。

__getitem__(self, key) 和 __setitem__(self, key, value)

使得对象可以使用[]进行获取和设置，就类似于字典一样。

__getslice__(self, begin, end) 和 __setslice__(self, begin, end, value)

使得对象可以使用[begin: end]这样类似list的方式进行获取和设置。

__iter__(self)

通过在函数内返回迭代器，使得对象可以被循环。

__add__(self, other)

实现加法操作。对于其他运算也有相应的魔法方法

__lt__(self, other)

x<y相当于调用x.lt(y)，定义了使用<时的评判标准。对于其他比较运算，也有对应的魔法方法。
